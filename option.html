CSS OPTIONS

Multi-column layout

Multi-column layout located in the option stylesheet1_5..css, line number 104 under 2nd responsive media query for swapping a single paragraph to 2 columns. View page in the secordary page(s)(1-5) option1.

Horizontal scrolling (please test on a phone)

The horizontal scrolling bar can be viewed on the home page. The horizontal scrolling bar code is located in the option stylesheet..css, line 219 under 2nd Responsive media query layout Comment for reading text horizontal across.

CSS attribute and/or substring matching attribute selectors

CSS attribute(s) is located in the option stylesheet, line 22, 40 and 200. The css attributes is used as a combined selector in .main, .css tags and target [color(white)] hyperlink text for the linked text HELLO on the image in the div container. View css attributes selector in the home page. and on the option.html page in the main heading tag ASIA NIGHTLIFE as a hyperlink.

@font-face rule

@font-face rule located in option stylesheet and option1_5.css stylesheet on line 7. Used as a font selection for all paragraph text and headers throughout home page and extended pages..

Alpha transparency (requires using HSL or RGB rather than HEX for color values)

You can view 5 different alpha transparency RGB colors in the home page on the image box as an overlay. THe alpha transparency rgba code is located in all 5 overlay sections, background-color property from line 112, 125, 138, 151, and 164. Code can be viewed in the option stylesheet

Text and box shadows

Box shadows property is located in option1_5.css stylesheet, line 57 under comment /*Content for image properies and shadowing*/. The Box shadow is used for shadowing the image background. View the example in option1.
